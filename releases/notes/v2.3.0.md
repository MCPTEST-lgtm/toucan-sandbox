# v2.3.0 Release Notes

Toucan 2.3.0 focuses on faster load times, streamlined authentication, and improved reliability. This release introduces OAuth2 login, optimizes dashboard loading, addresses a CSV export edge case, and includes a database schema migration to support upcoming capabilities.

## Highlights
- Add OAuth2 login (#128)
- Improve dashboard loading (#131)

## Bug Fixes
- Fix CSV export edge case (#139)

## Technical Notes
- db schema migration 2025_11_20: apply database migrations prior to deploying v2.3.0. This migration prepares the schema for new features. Review migration plan and ensure backups are in place before rollout.
- No breaking API changes expected in this release.
