# Contributing to toucan-sandbox

Thank you for your interest in contributing to this repository. Please follow these steps so your contribution can be reviewed and merged quickly.

1. Fork the repository and clone your fork locally.

 git clone https://github.com/<your-username>/toucan-sandbox.git
 cd toucan-sandbox

2. Create a branch for your change using the naming convention:

 feature/<short-description>-YYYYMMDD

 Example: feature/fix-readme-20251118

3. Make focused commits with clear commit messages. Use imperative mood, e.g., "Add unit tests for X".

4. Run tests and linters before pushing. If there are test scripts, run:

 npm test # or the project's test command

5. Push your branch to your fork and open a pull request against MCPTEST-lgtm/toucan-sandbox:main.

6. In your PR description, explain the change, any migration steps, and note if it is a breaking change.

7. Be responsive to review feedback; update your branch as requested.

Thank you!
