# GHSA Response: lodash and OpenSSL upgrades (2025-11-21)

This security advisory response mitigates the following vulnerabilities and upgrades the affected components to patched versions.

- CVEs addressed: CVE-2025-1111, CVE-2025-1452
- Affected paths:
  - /src/server/package.json (lodash)
  - /infra/Dockerfile (OpenSSL)
- Required upgrades:
  - lodash: 4.17.21
  - OpenSSL: 3.0.15

Mitigation overview:
- Bump lodash dependency in the application server to 4.17.21.
- Ensure the container image (or base OS layer) provides OpenSSL 3.0.15.

See mitigation matrix for concise before/after versions: ../mitigation_matrix.json
